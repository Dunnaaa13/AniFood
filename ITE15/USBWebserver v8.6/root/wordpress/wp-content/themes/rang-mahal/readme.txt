== Description ==
This theme extends theme "WP Bootstrap 4" with detailed customization. "Rang Mahal" is another easy to use WordPress theme. It is a child theme to world's most popular framework for responsive websites, Twitter Bootstrap 4 based WordPress theme WP Bootstrap 4.

== Installation ==
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.



== Frequently Asked Questions ==
= Does this theme support any plugins? =
Rang Mahal includes support for Infinite Scroll, Image gallery, Contact Form in Jetpack. It also supports all major page builder plugins like Beaver Builder, SiteOrigin and Elementor. This theme also supports contact  form plugins like Contact Form 7, Caldera Forms etc.

= Does this theme support multi-level navigation in main menu? =
Partially yes. This theme supports only two level of navigation. Third level child will not be displayed properly.
